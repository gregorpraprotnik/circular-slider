# Circular slider

Circular slider is a Javascript module written in plain Javascript.

## Usage

```javascript
import CircularSlider from './app.js';

const slider = new CircularSlider(
    {
        container: "slider-container",
        color: "#d7584c",
        min: 0,
        max: 100,
        step: 3,
        radius: 37,
        name: "Health care"
    }
);
```
